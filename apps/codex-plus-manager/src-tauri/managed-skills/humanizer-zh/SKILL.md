# Humanizer Zh

Rewrite Chinese text so it sounds like a real person wrote it. Preserve the user's meaning, facts, numbers, names, structure requirements, and target audience.

## Workflow

1. Identify the text type: chat message, article, marketing copy, tutorial, product copy, business document, or social post.
2. Remove common AI traces: stiff transitions, repetitive sentence rhythm, excessive summary language, generic adjectives, over-balanced phrasing, and unnatural enumerations.
3. Keep the language concrete and locally natural. Prefer short sentences when the original is conversational, and keep formal wording only when the document context requires it.
4. Do not add unsupported facts, claims, examples, citations, or promises.
5. If the user gives a target tone, follow it. If not, default to clear, natural, and slightly concise Chinese.

## Output

Return only the polished Chinese text unless the user asks for comparison or explanation.
